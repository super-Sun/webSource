Promise详解

是什么？

ES6提供的对象，就是一个对象，用来传递异步操作的消息。用于管理与异步 API 的交互的抽象对象

有什么用？

这是用于管理与异步 API 的交互的抽象对象

怎么用？

var promise = new Promise(function(resolve, reject) { /*code*/ });

参数:

promise :必需。承诺分配到的变量名称。

resolve :必需。一个函数，运行该函数可指示承诺已成功完成。

reject :可选。一个函数，运行该函数可指示出现错误，承诺已被拒绝。

基本的 api:

Promise.resolve(): 完成，并标记 成功

Promise.reject()：完成，并标记 失败

Promise.prototype.then()：回调

Promise.prototype.catch()：异常捕捉

var p = Promise.all([p1,p2,p3]); // p:所有的完成

Promise.race() // 竞速，完成一个即可
